# نبض (Nabd)

نظام ذكي لدعم التقييم النفسي للأطفال المتضررين في الحروب باستخدام تحليل السلوك غير اللفظي والصوت.

## التقنيات
- React 19 + TypeScript
- Vite + Tailwind CSS v4
- Google Gemini AI

## التشغيل
1. `npm install`
2. نسخ `.env.example` إلى `.env.local` وإضافة `GEMINI_API_KEY`
3. `npm run dev`
4. فتح `http://localhost:3000`
